    Do you need a blockchain?

    # 摘要

    该论文提出了一种结构化方法，已确定特定问题的合适的解决方案。主要深度分析了在供应链管理(Supply Chain Maangement)、银行和国际支付、以及去中心自治组织三个方向的案例。

    # 引言

    由于区块链技术所具有的特性，已经在金融交易、分布式云储存、智能财产、物联网、供应链管理、医疗保健、所有权和版权分配，以及去中心化自治组织等行业获得了很多的关注。 不受限制的区块链允许任何人的加入。受限的区块链是和中心化的数据库相似的。

    # 区块链背景

    定义writer是能够对数据库进行写操作的实体。定义reader虽然不能通过写操作来延长区块链但是可以读取，分析以及审查区块链。

    # 概念

    Permissionless Blockchains。(像Bitcoin和Ethereum)，任何节点可以任何时间接入和离开该网络。

    Permissioned Blockchains: 只有授权的节点，才能对区块链进行相应的读或写操作。

    Public Verifiability: 允许任何人可以验证该系统状态的正确性。

    Transparency:

    Privacy: 该特性是和透明性要有一个权衡的。

    Integrity： 该特性可却保证没有被授权的信息不会被修改。即数据是正确的。信息的完整性是和公共验证性密切相关的。

    Redundancy: 在区块链中，Redundancy是被所有writer通过复制提供的。

    Trust Anchor: 定义了谁拥有最高的权限。

    # 透明性和隐私之间的权衡

    如果区块链可以提供重要的隐私保障又可以系统的整个状态转移过程透明。隐私特性在公共的系统能够使用加密技术实现，但是会以较低的效率为代价。

    # 区块链在哪里会有意义？

    只有当多个互相不信任的实体想要交互并且改变系统的状态，并且不愿意使用在线第三方(TTP)时，使用开放或者许可的区块链才有意义。为了简化该过程，该论文提出了一个流程图：

    # 逐案处理

    主要概述了供应链管理和银行间支付案例。

    # 供应链管理

    由于这种复杂性，管理库存、流程和故障检测的相关成本特别昂贵。有几个公司想要提供基于区块链的解决方案，以提高供应链管理的效率。有一些人甚至声称区块链技术为需求而不是供应链铺平了道路。

    传统的SCM是由计划和沟通驱动的。公司决定什么产品在什么时候投放到市场，客户间接推动需求。

    在需求链管理(DCM)中，客户的利益是链的核心，降低成本，提供高性能的客户服务，以及更快的从想法到落实走向市场。

    供应链管理存在显示世界和数字世界之间存在问题。例如： 依赖于温度数据的智能合约必须确保用于签署数据的密钥是属于当前装运的传感器。客户仍然需要完全信任供应商的可靠数据，因为即使使用防篡改硬件，这种传感器也很容易被欺骗。一个很简单的场景如下图所示：

    # 银行间和国际支付

    对于传统的银行系统，对于银行间的货币转账如果没有一个代理银行，则可能需要额外的中间跳转，这会增加复杂性，延迟并增加成本。总而言之，代理银行系统的主要缺点是交易确认时间长，多个中间跳转成本增加，并且银行之间是要互相信任的。

    由于代理银行系统带来的高成本，许多人寄希望于分布式账本技术，以简化银行间支付。 在MAS的解决方案中，银行向MAS存入一定数量的资金，对应在分布式账本上获得相同的金额。

    Ripple旨在提供基于分布式账本的全球结算网络。如果所需外币低利率可用，银行可以继续使用代理银行处理付款。支付过程具有原子性。Ripple提供自己的货币XPR，可以作为交易的中间货币。

    具有公共验证性可以增加公众对货币系统的信任

    # 智能合约

    智能合约是一种自我执行的数字合约，或使其违约成本高昂。

    如果合约的细节希望被隐藏并且仅设计有限的已知参与者，则Hyperledger Fabric等许可的区块链是最合适的。

    目前关于智能合约的法律还不够完善。

    # 去中心化的自治组织

    去中心化自治组织(DAO)是通过一组智能合约自主运行的组织。

    在大多数情况下，DAOs不需要自己的区块链，而是更适合在现有的区块链之上使用现有的货币构建。

    # 知识产权所有权证明

    如果某个数字对象的创建者后来想要证明所有权，他可以使用公共区块链作为时间戳服务，通过将其与身份一起提交给该数字对象。也就是用一个哈希，在区块链上发布那个证明。可信的第三方也可以提供所有权证明(专利局)。

    分布式账本技术可以确保土地所有权不会以未被察觉的方式改变土地所有权。此外在区块链上土地所有权可以通过智能合约进行买卖。

    # 贸易和公平的交易协议

    在没有可信的争议调解人的情况下，数字商品的交易可行的，而在发生争议时，实物商品交易仍然需要可信的第三方。如果只交易数字产品，则无许可区块链可能是最合适的。
