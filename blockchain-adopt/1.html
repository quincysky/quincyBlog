    When Do We Need the Blockchain?

    # 摘要

    这篇论文主要回答了"When do we use the blockchain？"， "Where do use use the blockchain?", "Why do we use the blockchian?"这个三个问题。

    # 适用性和不适用性

    区块链的不适用场景就是根据区块链的特性(去中心化, 不可篡改, 可追溯, 透明, 匿名)总结出来的。列举一些不适用区块链场景的例子：

    ①如果想要开发频繁修改数据的系统，不适用搭配区块链存储数据。

    ②如果使用该系统的用户很少，也不适合通过区块链开发。区块链是为分布式网络开发，用户少意味着peer节点很少，效率会很低。

    ③对于一些需要一定程度的用户和数据隐私的社交媒体应用程序也是不适合的

    类似的还有很多。可以根据论文中所给图片来进行判别所开发系统到底适不适配区块链

    # 区块链的挑战

    区块链的局限性：可扩展性， 吞吐量， 延迟， 大小和带宽。这些局限都是目前比较热门的研究课题。

    传统的区块链是不能确保数据的隐私的，随着越来越多人的研究，可以使用on-chain和off-chain技术确保数据的隐私。

    区块链也是适用于物联网的。需要根据需求重新开发。
